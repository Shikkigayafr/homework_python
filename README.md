## 📋 Описание заданий

### Тема 1. Основы Python (задания в отдельных файлах)

| Файл | Описание |
|------|----------|
| `z1_name_age.py` | Проверка имени (буквы, длина 2–30) и возраста (≥ 0) |
| `z2_date_check.py` | Проверка даты в формате ДД.ММ.ГГГГ с учётом високосности |
| `z3_guess_game.py` | Игра «Угадай число» за 7 попыток |

### Тема 2. Работа со списками и алгоритмами

| Файл | Описание |
|------|----------|
| `job_spisok_algo_v1_z1.py` | Генерация N случайных чисел (1–100). Вывод: сумма, кол-во чётных, максимум |
| `job_spisok_algo_v1_z2.py` | Построение частотной таблицы символов (без словарей) |
| `job_spisok_algo_v1_z3.py` | Поиск простых чисел-палиндромов до N. Вывод среднего, запись в `palindromes.txt` |

### Тема 3. Комбинированная обработка текста и чисел

| Файл | Описание |
|------|----------|
| `komb_obrabotka_txt_and_chisel_v1_z1.py` | Обработка текстовых файлов |
| `komb_obrabotka_txt_and_chisel_v1_z2.py` | Работа с числовыми данными |
| `komb_obrabotka_txt_and_chisel_v1_z3.py` | Комплексная обработка данных |
